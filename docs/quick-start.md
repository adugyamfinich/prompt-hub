# Quick Start Guide

Get up and running with PromptHub in 5 minutes!

## üéØ What is PromptHub?

PromptHub is a collection of high-quality, tested prompts for AI models like Claude, GPT, and Gemini. Think of it as a recipe book for AI interactions.

## üöÄ Using a Prompt (3 Steps)

### Step 1: Find a Prompt

**Browse by what you need:**
- [Coding](../prompts/by-category/coding/) - Debug, generate, review code
- [Writing](../prompts/by-category/writing/) - Articles, emails, creative content
- [Analysis](../prompts/by-category/analysis/) - Data interpretation, research
- [Creative](../prompts/by-category/creative/) - Stories, art, brainstorming
- [Education](../prompts/by-category/education/) - Teaching, learning, explaining
- [Research](../prompts/by-category/research/) - Literature review, hypothesis

**Or search by keyword:**
```
Use GitHub search: #python #debugging
```

### Step 2: Customize the Prompt

Each prompt has variables marked like `{THIS}`. Replace them with your specific needs.

**Example:**

Original prompt:
```
You are a {EXPERTISE} expert. Help me {TASK} by {METHOD}.
```

Your version:
```
You are a Python expert. Help me debug my code by explaining each error.
```

### Step 3: Use It

Copy your customized prompt and paste it into:
- Claude (claude.ai)
- ChatGPT (chat.openai.com)
- Gemini (gemini.google.com)
- Any compatible AI tool

## üìñ Reading a Prompt File

Each prompt follows this structure:

```markdown
# Prompt Name

## Metadata
Quick info: category, difficulty, which AI models work

## Description
What this prompt does

## Use Case
When to use this prompt

## The Prompt
[THE ACTUAL PROMPT TO COPY]

## Variables to Customize
Explanation of {VARIABLES}

## Example Input/Output
Real examples showing it in action

## Performance Notes
Strengths, limitations, tips
```
